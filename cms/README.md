## useful command:

- `ghost stop` to stop Ghost
- `ghost start` to start Ghost
- `ghost log` views logs
- `ghost ls` to list all running Ghost blogs

## Node version

- currently using Nodejs V12.22.1

## install steps

### local development

1. cd ./cms
2. mkdir ./ghost
3. cd ./ghost
4. `ghost install version --local` // current version is 4.27.1
